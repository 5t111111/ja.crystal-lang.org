# 式としての if

`if` は式としても機能し、その値は、該当する分岐節の最後の式の値になります。

```ruby
a = if 2 > 1
      3
    else
      4
    end
a #=> 3
```

もし、条件に合致する分岐節が空の場合、もしくは合致する分岐節自体が存在しない場合には、そこに `nil` が与えられていたのと同じ挙動になります。

```ruby
# 例1
if 1 > 2
  3
end

# 上記は以下と同じ
if 1 > 2
  3
else
  nil
end

# 例2
if 1 > 2
else
  3
end

# 上記は以下と同じ
if 1 > 2
  nil
else
  3
end
```
